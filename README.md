# Guitarra Brasileira - Roadmap de Estudos

## Objetivo
Desenvolver domínio técnico, teórico e criativo da guitarra com foco em música brasileira, especialmente linguagem nordestina e pernambucana.

## Como usar
- Siga o ROADMAP.md
- Registre seu progresso em PRACTICE_LOG.md
- Estude diariamente com base em ROTINA.md
